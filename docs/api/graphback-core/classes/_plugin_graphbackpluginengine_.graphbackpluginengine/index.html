Version: 0.16.x

GraphbackPluginEngine

Allows to execute chain of plugins that create resources. Common use case is to decorate GraphQL schema with additional actions and generate files like resolvers and database access logic

Usage:

const engine = GraphbackPluginEngine({ schema });
engine.registerPlugin(plugin);
printSchema(engine.createResources());
